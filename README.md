# Study-Buddy-App
<h3>DataBase use : Firebase</h3>
<h3>Features : <h3>
1. Login/Registeration of Teacher & Student.<br>
2. Teacher can send Notices to Students.<br>
3. Teacher can upload Time table for perticular division.<br>
4. Real-time Notices.<br>
