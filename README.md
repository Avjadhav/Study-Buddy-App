# Study-Buddy-App
<h3>DataBase use : Firebase</h3>
Features : 
1. Login/Registeration of Teacher & Student.
2. Teacher can send Notices to Students.
3. Teacher can upload Time table for perticular division.
4. Real-time Notices.
